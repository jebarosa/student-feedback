// HOW AIML IS USED

AI & ML Inside This Portal

🧠

Sentiment Analysis

Your feedback text is run through an NLP model that classifies emotions as Positive, Neutral, or Negative using a trained Naive Bayes classifier.

📊

Feedback Clustering

K-Means clustering groups similar feedback responses, helping instructors identify common themes without reading every submission.

🔮

Trend Prediction

A time-series regression model predicts satisfaction trends over semesters, flagging potential drops before they become problems.

💡

Smart Suggestions

Based on historical data, the system auto-suggests improvement areas to faculty using a recommendation engine.
